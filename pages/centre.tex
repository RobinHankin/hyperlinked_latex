  \section{Centre help here}

The {\bf centre} $\centremath(L)$ of a Lie algebra $L$ is the set of
elements which have zero Lie bracket with any element of $L$.

\mainbox{Formally,}{Z(L) = \left\{x\in L\colon\bracket{x}{y}=0\quad\mbox{for all}\quad y\in L\right\}}


\subsection*{Facts}

\begin{itemize}
\item The centre is a good example of an \ideal.
\end{itemize}
