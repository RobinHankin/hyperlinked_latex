  \section{Matrix logarithm help here}

If $M$ is a square matrix with real or complex entries, then the {\bf
  matrix logarithm} of $M$, written $\matlogmath M$ is an inverse of
the \matexp\ function.


\boxeq{
\log M=\sum_{r=1}^\infty(-1)^{r+1}\frac{(M-I)^r}{r}
}

If $M\in M_c(\mathbb{C})$ and both $\left|\left|M\right|\right|<\log 2$
and $\left|\left|\exp M-I\right|\right|<1$, then $\matlogmath e^M=M$.


\subsection*{Facts}
\begin{itemize}
  \item Every invertible $n\times n$ complex matrix can be expressed
    as $e^X$ for some $X\in M_n(\mathbb{C})$.
    \end{itemize}
